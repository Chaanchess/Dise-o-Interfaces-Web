Columnas múltiples en CSS

Permite una fácil definición de múltiples columnas de texto

¿Qué es? , para que sirve y compatibilidad.

Column es una propiedad de CSS3, que nos ofrece la posibilidad de trabajar con las columnas y darles estilos a las mismas, aquí abajo dejo una captura de canIuse y su link correspondiente para ver su compatibilidad.

-- > Link CanIuse <---

Column-Count

Especifíca el numero de columnas en las cuales va a dividir el contenido.

    Posibles valores

  • Un numero (x).
  • Automático (auto).
  • Valor initial (initial).
  • Valor inherit (inherit).

(Código css)

En mi caso lo aplico a una etiqueta de párrafo.

#columnCount{
      column-count: 3;
      padding: 3px;
 }

Resultado tras aplicar un colum-count:3.

Column-gap

Especifíca la distancia entre las columnas.

    Posibles medidas

  • En pixeles (px).
  • normal (normal).
  • Valor initial (initial).
  • Valor inherit (inherit).

(Código css)

#columnGap{
      column-count: 3;
      column-gap: 80px;
 }

Separación de columnas aplicando un column-gap:80px.

Column-rules-style

Delimita con una línea la distancia entre columnas.

    Posibles valores

  • Por defeccto (none)
  • Oculto (hidden)
  • Estilo dotted (dotted)
  • Estilo dashed (dashed)
  • Estilo solido (solid)
  • Estilo doble (double)
  • Estilo initial (initial)
  • Estilo inherit (inherit)

(Código css, para aplicar un estilo a la línea)

#columnRules{
      column-count: 3;
      column-gap: 80px;
      column-rule-style: solid;
 }

Column-rules-width

Aumentando el ancho del delimitador de columnas.

    Posibles medidas

  • En pixeles (px).
  • En porcentaje (%).
  • Valor relativo de pantalla (vh).

(Código css, para aumentar el ancho del delimitador)

#columnRules{
      column-count: 3;
      column-gap: 80px;
      column-rule-style: solid;
      column-rule-width: 15px;
 }

Column-rules-color

Cambia el color del delimitador de columnas column-color.

    Notaciones de color posibles.

  • Valores RGB.
  • Valores Hexadecimal.
  • Valores HSL.
  • Valores RGBA.
  • Valores HSLA.

    • *Para más información de uso de colores acceder a apartado correspondiente.*

(Código css, para aplicar un ancho a la línea)

#columnRuleColor{
      column-count: 3;
      column-gap: 80px;
      column-rule-style: solid;
      column-rule-width: 15px;
      column-rule-color: red;
 }

Column-rules (formato reducido)

Aplicando todas las propiedades colum-rules de forma reducida.

    Formato forma reducida

  • column-rule: (tamaño) (tipo de borde) (color)

(Código css, aplicando todas las propiedades colum-rules de forma reducida)

#columnRuleColor{
      column-count: 3;
      column-gap: 80px;
      column-rule: 15px solid red;
 }

Colum-span

Especifica cuantas columnas va a ocupar el elemento.

    Posibles valores

  • Valor por defecto (none).
  • Ocupa todas las (all).
  • Valor initial (initial).
  • Valor inherit (inherit).

(Código css, aplicando un span)

#columnRuleColor{
      column-gap: 40px;
      margin: 1em auto;
      color: blue;
      column-span: inherit;
 }

Colum-width

Especifíca el ancho asignado a las columnas.

    Posibles medidas

  • Automático (auto).
  • En pixeles (px).
  • Valor initial (initial).
  • Valor inherit (inherit).

(Código css, aplicando un ancho a las columnas)

#columWidth{
      column-count: 4;
      column-gap: 20px;
      column-width: 600px;
 }
