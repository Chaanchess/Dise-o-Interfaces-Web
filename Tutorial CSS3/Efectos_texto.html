Tutorial Fuentes y Ef Texto

En este tutorial explicaremos todas las propiedades de las fuentes de texto.

Font-face

La propriedad @font-face permite al autor especificar fuentes online para visualizar en sus páginas web. Al permitir a los autores proporcionar sus propias fuentes, @font-face elimina la necesidad de depender del numero limitado de fuentes de usuarios instaladas en sus ordenadores.

Sintaxis:

@font-face {
 font-family: 'un-nombre-de-fuente-remota';
 src: 'origen' [,origen]*;
 [font-weight: 'peso'];
 [font-style: 'estilo'];
}

Ejemplo:

@font-face {
 font-family: "Mystery Quest";
 src: local('MysteryQuest'),
   url("../fonts/MysteryQuest-Regular.ttf");
}

Se puede especificar una fuente almacenada en el ordenador del usuario utilizando la sintaxis local(). Si esa fuente no es encontrada, se intentarán otros orígenes hasta encontrar una fuente.

Font-kerning

La propriedad font-kerning define cómo se espacian las letras. Esta característica hace que el espaciado entre caracteres sea más uniforme y agradable de leer de lo que sería.

Posibles valores: auto/normal/none.

auto: El navegador determina si se debe usar o no el kerning de fuentes.

normal: Se debe aplicar la información de kerning de fuentes almacenada en la fuente.

none: La información de kerning de fuentes almacenada en la fuente está deshabilitada.

Posibles valores:
font-kerning: auto;
font-kerning: normal;
font-kerning: none;

Ejemplo:

{
 font-kerning: none;
}

Text-orientation

La propiedad CSS text-orientation define la orientación del texto en una línea de escritura. Ésta propiedad CSS es util para controlar la forma en que se muestran los lenguajes que utilizan escritura vertical, y tambien para construir encabezados verticales para tablas.

Ejemplo:

{
 writing-mode: vertical-rl;
 text-orientation: upright;
}

Font-size-adjust

La propiedad font-size-adjust especifica que el tamaño de la fuente debería escogerse basándose en el tamaño de las minúsculas en lugar las mayúsculas.

Ejemplo:

{
 font-size-adjust: 0.58;
}

Text-rendering

La propiedad Text-rendering básicamente toma una decisión sobre lo que el navegador debe priorizar cuando se procesa texto. Puede optimizar en términos de velocidad (optimizeSpeed), precisión geométrica (geometricPrecision), y por supuesto, legibilidad (optimizeLegibility)

Posibles valores:

text-rendering: optimizeSpeed;
text-rendering: optimizeLegibility;

Font-stretch

Algunas familias de fuentes ofrecen caras adicionales en las que los caracteres son más estrechos que la cara normal (caras condensadas) o más anchos que la cara normal (caras expandidas). Puede usar el Font-stretch para seleccionar una cara condensada o expandida de dichas fuentes. Si la fuente que está utilizando no ofrece caras condensadas o expandidas, esta propiedad no tiene efecto.

Para los navegadores con motor de chrome(Blink) es necesario indicar la propiedad dentro de la declaración de la font-family (font-stretch: 1% 500%;) 

                                        @font-face {
                                            src: url('../fonts/LeagueMonoVariable.ttf');
                                            font-family:'LeagueMonoVariable';
                                            font-style: normal;
                                            font-stretch: 1% 500%; /* Required by Chrome */
                                          }

font-stretch: normal;

font-stretch: expanded;
font-stretch: extra-expanded;
font-stretch: ultra-expanded;
font-stretch: 0%;
font-stretch: 50%;
font-stretch: 200%;

text-fill-color

La propiedad CSS text-fill-color especifica el color de relleno de las letras de un texto. Si la propiedad no está establecidad se usará el valor de la propiedad color.

Solo está suportada utilizando el modificador -webkit-

Ejemplo:

{
 font: bold 5.8em Bebas, Arial;
 margin: 0.4em;
 background: url(../img/bg.jpg) center;
 -webkit-background-clip: text;
 -webkit-text-fill-color: transparent;
 text-transform: uppercase;
}

Text-shadow

La propiedad text-shadow agrega sombra al texto. Acepta una lista de sombras separadas por coma, para aplicarlas al texto y a la propiedad text-decorations del elemento. Cada sombra es especificada como un offset del texto, junto con valores de color y radio de difuminación opcionales. Múltiples sombras son aplicadas del frente al fondo, en el mismo orden que sean definidas.

Atributos:
text-shadow: desplazamiento eje-x | desplazamiento eje-y | valor de difuminado | color

Ejemplo:

{
 font: bold 3em Arial;
 text-shadow: 0 1px 0 #ccc,
 0 2px 0 #c5c5c5,
 0 3px 0 #bbb,
 0 4px 0 #b5b5b5,
 0 5px 0 #aaa,
 0 6px 0 #a5a5a5,
 0 7px 3px rgba(0, 0, 0, 0.3),
 0 10px 10px rgba(0, 0, 0, 0.2),
 0 10px 20px rgba(0, 0, 0, 0.15),
 0 20px 20px rgba(0, 0, 0, 0.1);
 color: rgba(255, 255, 255, 0.8);
}
