Fondos

El fondo es el decorado que se encuentra en la primera capa de nuestra página web, sobre él colocamos los elementos de la web y con las siguientes propiedades podemos controlar la forma en la que se presenta el fondo

Propiedad "background-color"

Esta propiedad establece el color de un elemento. El color puede especificarse de varias formas:

• En RGB:

Código del ejemplo 

                                #backgroundColorRgb {
                                    rgb(227, 127, 107);
                                    width: 300px;
                                    height: 90px;
                                }
Resultado:



• En RGBA:

Código del ejemplo 

                            #backgroundColorRgba {
                                background-color: rgba(70, 88, 192, 0.4);
                                width: 300px;
                                height: 90px;
                            }
Resultado:



• En código hexadecimal:

Código del ejemplo 

                                #backgroundColorHex {
                                    background-color: #112f50;
                                    width: 300px;
                                    height: 90px;
                                }
Resultado:



• Indicando directamente el nombre del color:

Código del ejemplo 

                                #backgroundColorNombre {
                                    background-color: violet;
                                    width: 300px;
                                    height: 90px;
                                }
Resultado:



• Con el modelo hsl:

Código del ejemplo 

                                #backgroundColorHsl {
                                    background-color: hsl(10, 90%, 56%);
                                    width: 300px;
                                    height: 90px;
                                }
Resultado:



• Con el modelo hsla:

Código del ejemplo: 

                                #backgroundColorHsla {
                                    background-color: hsla(10, 90%, 56%, 0.3);
                                    width: 300px;
                                    height: 90px;
                                }
Resultado:

Propiedad "background-image"

Define una imagen como fondo de un elemento.
Para poner una imagen, hay que añadir en el archivo css la ruta relativa, absoluta, o bien, una url externa de la imagen que queramos.

Código del ejemplo 

                                    #fondoImagen{
                                        background-image: url(../img/fondo.jpg);                        
                                        width: 700px;
                                        height: 300px;
                                        border: 1px black solid;
                                        background-size: 720px 390px;
                                    }
Resultado:

Para poner varias imágenes de fondo tenemos que separar las url por comas. El orden de las url es importante, los fondos estarán unos sobre otros ordenados de más abajo a más arriba.

Código del ejemplo 

                                    #fondoVariasImgs {
                                        background-image: url("../img/fondo_pajaros.png "), 
                                            url("../img/cielo.jpg ");                                
                                        width: 700px;
                                        height: 300px;
                                        border: 1px black solid;
                                        background-size: 720px 390px;
                                    }
Resultado

También podemos definir como fondo de imagen un degradado lineal.
Para profundizar sobre los degradados: Documentación degradados

Código del ejemplo 

                                    #fondoDegradado {
                                        background-image: linear-gradient(gray, black);
                                        width: 720px;
                                        height: 390px;
                                    }
Resultado:

Consta de 3 propiedades:

  • none: es la propiedad por defecto, no se mostrará ninguna imagen de fondo.
  • initial: establece la propiedad a su valor predeterminado.
  • inherit: la propiedad hereda del elemento padre.

Propiedad "background-repeat "

Especifica si el fondo se repite y de qué manera. Admite los siguientes valores:

• repeat: el fondo se repite horizontal y verticalmente. Es la que se le aplica por defecto.

Código del ejemplo 

                                    #backgroundRepeat {
                                        background-repeat: repeat;
                                        width: 700px;
                                        height: 300px;
                                        border: 1px black solid;
                                        background-image: url(../img/fondo.jpg);
                                    }
Resultado:



• repeat-x: el fondo se repite en horizontal.

Código del ejemplo 

                                    #backgroundRepeatX {
                                        background-repeat: repeat-x;
                                        width: 300px;
                                        height: 200px;
                                        border: 1px black solid;
                                        background-image: url(../img/fondo.jpg);
                                        background-size: 100px 100px;
                                    }
Resultado:



• repeat-y: el fondo se repite en vertical.

Código del ejemplo 

                                    #backgroundRepeatY {
                                        background-repeat: repeat-y;
                                        width: 300px;
                                        height: 200px;
                                        border: 1px black solid;
                                        background-image: url(../images/fondo.jpg);
                                        background-size: 100px 100px;
                                    }
Resultado:



• no-repeat: el fondo no se repite.

Código del ejemplo: 

                                    #backgroundNoRepeat {
                                        background-repeat: no-repeat;
                                        width: 300px;
                                        height: 200px;
                                        border: 1px black solid;
                                        background-image: url(../images/fondo.jpg);
                                        background-size: 100px 100px;
                                    }
Resultado:

                                    margin: 5px 0px 5px 0px;

                                    height: 300px;

Propiedad "background-position"

La propiedad de posición de fondo establece la posición de inicio de una imagen de fondo.

De forma predeterminada, una imagen de fondo se coloca en la esquina superior izquierda de un elemento y se repite tanto vertical como horizontalmente.

Si solo especifica una palabra clave, el otro valor será "center"

Los posibles valores son:

  • left top
  • left center
  • left bottom
  • right top
  • right center
  • right bottom
  • center top
  • center center
  • center bottom
Código del ejemplo: 

                                #backgroundpositionleft{
                                    background-image: url('../img/backgroundVictorLogoCSS.png');
                                    background-repeat: no-repeat;
                                    background-position: right center; 
                                    border: 1px solid black;
                                    margin: 5px 0px 5px 0px;
                                }
Resultado:

%X %Y

El primer valor es la posición horizontal y el segundo valor es la vertical.


La esquina superior izquierda es 0% 0%. La esquina inferior derecha es 100% 100%.
Si solo especifica un valor, el otro valor será 50%.
El valor predeterminado es: 0% 0%

Propiedad "background-origin"

La propiedad de origen de fondo especifica dónde se coloca la imagen de fondo.

Nota: Si la propiedad background-attachment está establecida en "fixed", esta propiedad no tiene ningún efecto.

padding-box

Valor por defecto. La imagen de fondo comienza desde la esquina superior izquierda del borde de relleno

Código del ejemplo: 

                                                    #backgroundoriginpadding {
                                                        background-image: 
                                                        url('..//backgroundVictorLogoCSS.png');
                                                        background-origin: padding-box;
                                                        background-position: 100% 100%;
                                                        background-repeat: no-repeat;
                                                        border: 15px solid black;
                                                        margin: 5px 0px 5px 0px;
                                                        padding: 20px;
                                                    }
Resultado:

border-box

La imagen de fondo comienza desde la esquina superior izquierda del borde

Código del ejemplo: 

                                                    #backgroundoriginborder {
                                                        background-image: 
                                                        url('../img/backgroundVictorLogoCSS.png');
                                                        background-origin: border-box;
                                                        background-position: 100% 100%;
                                                        background-repeat: no-repeat;
                                                        border: 15px solid black;
                                                        margin: 5px 0px 5px 0px;
                                                        padding: 20px;
                                                    }
Resultado:

content-box

La imagen de fondo comienza desde la esquina superior izquierda del contenido

Código del ejemplo: 

                                                    #backgroundorigincontent {
                                                        background-image: 
                                                        url('../img/backgroundVictorLogoCSS.png');
                                                        background-origin: content-box;
                                                        background-position: 100% 100%;
                                                        background-repeat: no-repeat;
                                                        border: 15px solid black;
                                                        margin: 5px 0px 5px 0px;
                                                        padding: 20px;

El fondo se extiende completamente a través del elemento, incluso bajo el borde.

Código del ejemplo

                                                            border: 10px dotted black;
                                                            padding: 35px;
                                                            background:yellow;
                                                        }

Propiedad "background-size"

La propiedad background-size define el tamaño de la imagen de fondo.

BACKGROUND-SIZE: AUTO AUTO;

La imagen de fondo mantendrá su tamaño original por defecto, o si aplicamos el tamaño con auto.

Por ejemplo, esta imagen de fondo tiene un tamaño de 1024 x 835 px. Es mayor que su contenedor, de 150px de alto, por lo que se le aplica un clippeo que la extiende hasta el límite marcado por el borde

Código del ejemplo 

                                                #background-size-ejemplo1-aplicado {
                                                    background-size: auto auto;
                                                    background-image: 
                                                    url(../img/La%20Noche%20Estrellada.jpg);
                                                    background-repeat: no-repeat;
                                                    height: 150px;
                                                    padding: 0;
                                                    border: 10px dashed rgba(0, 0, 255, 0.37);
                                                }
Resultado:

BACKGROUND-SIZE: 120px 80px;

La imagen de fondo se adaptará al tamaño definido en píxels. El primer valor es el tamaño horizontal y el segundo el vertical.

Código del ejemplo 

                                                    #background-size-ejemplo2-aplicado {
                                                        background-size: 120px 80px;
                                                        background-image: 
                                                        url(../img/La%20Noche%20Estrellada.jpg);

                                                    }
Resultado:

BACKGROUND-SIZE: 50% 50%;

Se puede usar porcentajes como valores también. Precaución, esto puede alterar la relación de aspecto de la imagen de fondo, y llevar a resultados inesperados

Código del ejemplo

                                                        background-size: 50% 50%;

Propiedad "background-blend-mode"

La propiedad background-blend-mode define el modo de fusión de cada capa de fondo (color y / o imagen).

Los diferentes valores posibles son:

  • normal:Predeterminado. Establece el modo de fusión en normal
  • multiply: Establece el modo de fusión a multiplicar
  • screen: Establece el modo de fusión en la pantalla
  • overlay: Establece el modo de fusión para superponer
  • darken: Establece el modo de fusión para oscurecer
  • lighten: Establece el modo de fusión para aligerar
  • color-dodge: Establece el modo de fusión para esquivar el color
  • saturation: Establece el modo de fusión en saturación
  • color: Establece el modo de fusión para colorear
  • luminosity: Establece el modo de fusión en luminosidad.

Propiedad "background shorthand"

La propiedad abreviada background establece todas las propiedades del fondo en una sola declaración

Las propiedades que pueden ser establecidas, en ese orden, son:

  • background-color
  • background-image
  • background-position
  • background-size
  • background-repeat
  • background-origin
  • background-clip
  • background-attachment.

No importa si alguno de los valores falta, por ejemplo, background:#ff0000 url(imagen.gif); está permitido.

Nota: Internet Explorer 8 y versiones anteriores no soporta múltiples imágenes de fondo en un elemento.

Nota: Si una de las propiedades en la declaración abreviada es la propiedad bg-size, se debe usar una / (barra inclinada) para separarla de la propiedad bg-position, por ejemplo background:url(sonrisa.gif) 10px 20px/50px 50px; dará como resultado una imagen de fondo, colocada a 10 píxels desde la izquierda, 20 píxels desde la parte superior, y el tamaño de la imagen será de 50 píxels de ancho por 50 píxels de alto.

Nota: Si se usan múltiples fuentes como imágenes de fondo, pero también se desea un color de fondo, el parámetro de background-color debe ir al final en la lista
